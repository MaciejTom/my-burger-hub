
# My burger hub

This is a website of a burger restaurant with a blog, created in Gatsby. All content was uploaded using Wordpress and then pulled to Gatsby using GraphQL. Added burgers in wordpress, using ACF and CPT automatically appear in the restaurantâ€™s menu. Posting blog posts also occurs automatically.

## Tech Stack
- Gatsby
- React
- Wordpress
- GraphQL
- Advanced custom fields
- Custom post type
- Styled components

## Live demo

https://myburgerhub.gatsbyjs.io/

## How to run locally?

Clone the project

```bash
git clone https://github.com/MaciejTom/my-burger-hub.git
```

Go to the project directory

```bash
cd my-burger-hub
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
